AI Resume Builder

This project is a simple Generative AI application developed using Python and the Gemini API.

The application collects user information such as name, education, skills, projects, and career goals. Based on the input, the system generates a professional resume summary, skills section, and project description.

Technologies Used
- Python
- Google Colab
- Gemini API

Features
- Generates resume content automatically
- Demonstrates prompt-based text generation
